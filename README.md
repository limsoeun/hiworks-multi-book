# Hiworks에서 회의실을 다중으로 예약하기위한 프로그램

매일 같은 시간 회의를 진행하면서 같은 회의실을 하루씩 반복 예약하는 번거로움을 해결하기 위해 만들게 되었습니다.

*지정한 기간의 월~금요일에 [같은 회의실/같은 시간/같은 회의 예약 사유]로 예약됩니다.
